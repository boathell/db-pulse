# Copyright, Sources, and Responsible Use

This operational policy describes DB Pulse copyright, attribution, correction, and responsible-use boundaries; it is not legal advice.

The repository's MIT License covers project source code and original repository documentation unless a file says otherwise. It does not relicense third-party articles, papers, standards, product documentation, trademarks, feeds, or datasets.

DB Pulse stores controlled evidence metadata and canonical links. Public output does not intentionally reproduce full third-party articles, paywalled material, restricted datasets, raw collector payloads, or private operational data.

Facts, inference, editorial judgment, prediction, and opportunity hypotheses remain distinct. Confidence, heat, impact, and value scores are decision-support estimates, not objective truth or professional advice. Verify original evidence before procurement, architecture, investment, legal, compliance, or security decisions.

Report factual, attribution, copyright, privacy, or removal concerns with the [content correction form](https://github.com/boathell/db-pulse/issues/new?template=content-correction.yml). Cite a specific DB Pulse page and access date when referring to its synthesis; cite the original publisher whenever referring to an underlying fact.
