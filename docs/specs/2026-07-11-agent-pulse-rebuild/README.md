## 目标结果

Agent Pulse 不再是“AI 新闻列表”，而是一个可解释、可调控、可持续发布的行业认知流水线：

1. 采集一手事实、专家判断和跨平台热度信号。
2. 将同一事件的多条原始信号聚成一个事件。
3. 分别计算事实可信度、行业影响、扩散热度与业务价值。
4. 输出中文事实摘要、技术洞察、行业判断、未来观察点和业务价值。
5. 通过管理台审核、置顶、隐藏、改分和编辑。
6. 导出不含隐私信息的静态站点并自动部署到 GitHub Pages。
