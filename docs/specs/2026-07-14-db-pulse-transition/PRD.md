# PRD：DB Pulse 中国数据库行业认知与决策系统

## 产品定义

DB Pulse 面向 CEO、数据库从业者、DBA 和数据架构师，把中国数据库行业的官方发布、论文、资本动作、政策变化与社区信号，收敛为可验证事件、战略判断、下一观察和可执行动作。

## 首期边界

- 公开内容只包含 `database-cn`；海外数据库只作为国内事件的比较证据。
- 历史基线从 2022 年开始，覆盖产品与技术演进、生产验证、政策标准、商业采用和资本变化。
- 保留首页、趋势、事件脉络、角色、选型与成本、来源、Scout、产品方法和 Changelog 路由。
- 中文为事实基准；公开 Event 必须具有完整英文翻译。
- 热度只能来自真实传播证据；没有传播测量的历史事件热度为 0。

## 首批产品面

覆盖达梦、人大金仓、GBase、GoldenDB、OceanBase、TiDB、openGauss、GaussDB、PolarDB、TDSQL、Vastbase、SequoiaDB、MatrixOne、Apache Doris、StarRocks、TDengine、NebulaGraph 和 Milvus。

六条主线为数据库内核与架构、分布式/云原生/Serverless、实时分析/湖仓/多模数据、稳定性/安全/运维/成本、产品商业化与行业落地、国产生态/资本/政策标准。决策角色为 CEO、DBA、数据架构师和数据库从业者。

## 完成标准

- 公开快照、API、页面和搜索结果不包含旧 AI 行业事件。
- 至少 36 个双语 Event，18 个核心生态均有官方来源和至少一个 Event。
- 来源默认为 draft/shadow；只有通过既有治理门禁后才能 active。
- 选型与成本只展示可回链的部署、许可、兼容性和定价口径，不生成排名。
- 包、产品、Changelog 和站点身份重置为 DB Pulse 0.1.0。
